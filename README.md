# CyberSentinel AI

AI-Powered Security Report Analysis and Remediation Platform
Bachelor's Final Year Project — ICT University Cameroon

## Quick Start

### Terminal 1 — Backend
```bash
cd backend
.\venv\Scripts\activate
python app.py
# Runs on http://localhost:8000
# API docs at http://localhost:8000/docs
```

### Terminal 2 — Frontend
```bash
cd frontend
npm run dev
# Runs on http://localhost:5173
```
